# Agent Instructions

This project is an online yoga membership platform.

Before writing code, read:

- AI_CONTEXT.md
- PROJECT_RULES.md
- DEVELOPMENT_WORKFLOW.md
- DECISIONS.md
- TASKS.md

## Important

This is a Next.js 15 App Router project.

Use:
- TypeScript
- Tailwind CSS
- Shadcn UI
- Supabase
- Stripe
- Mux
- Resend
- Vercel

Do not use:
- Express
- Shopify
- Separate backend repo
- Microservices
- Public video URLs
- Client-side-only authorization

## Working Rules

- Build one task at a time.
- Keep diffs small.
- Do not refactor unrelated files.
- Do not install packages without asking.
- Do not create new architecture without updating DECISIONS.md.
- Follow TASKS.md sprint order.

## Next.js 15

This version may have breaking changes. When unsure, check local Next.js docs in `node_modules/next/dist/docs/` before implementing.
